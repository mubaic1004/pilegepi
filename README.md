# 皮了个皮

原创萌宠叠牌三消手机网页游戏。点击未被遮挡的牌放进七格卡槽，三张相同自动消除；第一关轻松教学，第二关为高随机、低通关率挑战。

## 本地运行

```powershell
npm install
npm run dev
```

## 测试与构建

```powershell
npm test
npm run test:difficulty
npm run build
npm run preview
```

生产文件输出到 `dist`，可直接部署到任意静态网站托管服务。

## 玩法

- 点击没有被上层遮挡的萌宠牌，牌会进入底部七格卡槽。
- 卡槽中凑齐三张相同牌会自动消除，七格塞满即失败。
- 第一关为18张牌的教学局，不会失败。
- 第二关包含四层中央牌塔和左右盲盒，可使用撤回、移出、洗牌各一次。
